# Strawberry Farm

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## GitHubとの連携

このリポジトリをGitHubに接続するには：

1. GitHubで新しいリポジトリを作成
2. 以下のコマンドを実行：

```bash
git remote add origin https://github.com/YOUR_USERNAME/strawberryfarm.git
git branch -M main
git push -u origin main
```
